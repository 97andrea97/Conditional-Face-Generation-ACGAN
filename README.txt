The notebook "ACGAN" and "GAN" contain respectively the codes to define and train the ACGAN and the GAN.
the notebook "generate images" uses the saved models "cgan_final_100epoch" or "gan_final_100_eoch" to generate faces.
the notebook "metric_proposal" contains a metric proposal, which is actualy a negative result, it is not a reliable metric.
csv file "list_attr_celeba" contains the attributes of the images in the celebA dataset which must be downloaded to run the notebooks for training.
